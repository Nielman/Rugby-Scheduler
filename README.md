
# Rugby Match Day Scheduler

This is a web app to manage and generate a randomized rugby match day schedule.

## 🏗️ Tech Stack

- React (Next.js)
- Tailwind CSS
- TypeScript
- Hosted on Vercel

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 📦 Deployment

Push to GitHub, then import repo into [Vercel](https://vercel.com/import) for instant deployment.
